JadInfo 1.0.1

Content plugin for show info from *.jad and *.jar files.
This is new independet version of MIDlet plugin (http://www.totalcmd.net/plugring/wdx_MIDlet.html).


*** Installing ***

Just enter to archive with plugin and TC ask you about plugin installing. 
Create new columns set (Configuration - Options - Custom columns - New).
Add new columns with jadinfo fields.
