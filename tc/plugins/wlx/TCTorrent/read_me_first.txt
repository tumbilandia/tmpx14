English

This archive contains two installers. 

"wlx_wdx_installer_x64.zip" is preferable, this include both wlx/wdx plugins in one file - but this meant that installer will install this file as lister plugin, and you need to install content part manually. If you have trouble with manual wdx installation, try separate automated installer "wdx_installer_x64.zip".
